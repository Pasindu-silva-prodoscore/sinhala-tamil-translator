# AGENTS.md — results/

## What This Directory Contains

All evaluation outputs. Generated by `src/evaluate.py` and the notebook.

```
results/
├── baseline_scores.json        ← {"system": "Baseline", "bleu": X, "chrf": Y, "n_sentences": N}
├── cascade_scores.json         ← {"system": "Cascade",  "bleu": X, "chrf": Y, "n_sentences": N}
├── comparison.md               ← Full Markdown report with scores + sample translations
├── score_comparison.png        ← Bar chart (BLEU + chrF, both systems)
├── baseline_loss.png           ← Training loss curve
└── labse_distribution.png      ← LaBSE similarity histogram
```

## Reporting Results in the Dissertation

The JSON score files contain the exact numbers to cite in Chapter 4 (Results) and Chapter 6 (Evaluation):

```python
import json
baseline = json.loads(Path('results/baseline_scores.json').read_text())
cascade  = json.loads(Path('results/cascade_scores.json').read_text())
print(f"Baseline BLEU: {baseline['bleu']}")
print(f"Cascade  BLEU: {cascade['bleu']}")
print(f"Delta:         {cascade['bleu'] - baseline['bleu']:+.2f}")
```

## Significance Testing

For the full dissertation, bootstrap resampling significance test is required (p < 0.05).
sacreBLEU provides this:

```python
import sacrebleu
bootstrap = sacrebleu.corpus_bleu(cascade_hyps, [references],
                                   bootstrap_resample=True)
```

Report significance in the dissertation as: "The cascade improvement is statistically significant (p < 0.05, bootstrap resampling, Koehn 2004)."

## Adding Human Evaluation Results

After collecting bilingual rater scores, add:

```python
human_results = {
    "n_sentences": 200,
    "raters": 3,
    "adequacy_mean": 3.8,
    "fluency_mean": 3.5,
    "cohens_kappa": 0.67,
    "system": "Cascade"
}
Path('results/human_eval_scores.json').write_text(
    json.dumps(human_results, indent=2), encoding='utf-8'
)
```

Cohen's Kappa above 0.6 is required for acceptable inter-annotator agreement (Landis & Koch 1977).
