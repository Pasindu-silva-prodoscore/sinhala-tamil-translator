# Results Directory — Agent Guide

## Files Generated Here

```
results/
├── baseline_scores.json        ← BLEU + chrF for the direct Transformer baseline
├── cascade_scores.json         ← BLEU + chrF for the cascade pivot system
├── comparison.md               ← Full comparison report with sample translations
├── score_comparison.png        ← Bar chart: BLEU and chrF side-by-side
├── baseline_loss.png           ← Training loss curve for baseline model
└── labse_distribution.png      ← Histogram of LaBSE similarity scores
```

## Score File Format

```json
{
  "system": "Cascade",
  "bleu": 15.42,
  "chrf": 38.71,
  "n_sentences": 20
}
```

## Interpreting Results

| BLEU Range | Interpretation |
|------------|----------------|
| < 10 | Barely any correct translation |
| 10–20 | Gist conveyed, many errors |
| 20–30 | Understandable, below practical usefulness threshold |
| 30+ | Practically useful (Papineni et al. 2002 threshold) |

**PoC targets:**
- Baseline: BLEU 3–8 (expected given only ~100 training pairs)
- Cascade: BLEU 12–22 (pretrained models, no fine-tuning)

**Full research targets:**
- Exceed BLEU 21.26 for Si→Ta (current SOTA, Nissanka et al. 2020)
- Exceed BLEU 18.05 for Ta→Si (current SOTA, Ranathunga et al. 2018)

## Comparison Report (comparison.md)

The report is generated by `src/evaluate.py` and includes:
1. Automatic scores table
2. BLEU/chrF delta (cascade − baseline)
3. 10 sample translations with columns: Source | Reference | Baseline | Cascade | Intermediate English
4. Error propagation analysis section

The intermediate English column is critical for Research Sub-question 2 on error propagation.

## Reproducibility

To reproduce results exactly:
- Test set is fixed with `seed=42` in `src/preprocess.py`
- BLEU computed via `sacrebleu.corpus_bleu()` (standardised tokenisation)
- All scores are reported on the same 20-sentence PoC test set (500-sentence in full system)

When publishing the full research results, include the `test_set/` files and these JSON score files so others can reproduce and compare against your numbers.
